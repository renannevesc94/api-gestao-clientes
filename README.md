##
Esta **API** tem como objetivo possibilitar o cadastro e gestão de clientes e licenças de uso de software de forma simplificada, a API fornece métodos e rotas para: **inclusão, exclusão, edição, e atualização de status dos clientes** alem disso a API fornece um método para fazer a **busca de status** do cliente passando apenas o CNPJ no query param, com isso sãõ retornadas duas strings, uma contendo o *status* do cliente e a outra contendo uma *mensagem* que pode ser usada para exibir um alerta para o cliente no caso de bloqueio ou atraso de pagamento por exemplo.
